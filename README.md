# cirunner

Manage config and installation of Gitlab CI runner

This module may be used with a simple `include ::cirunner`

# Compatibility

This module has been tested to work on the following systems with Puppet
versions v3, v3 with future parser and v4 with  Ruby versions 1.8.7 (Puppet v3
only), 1.9.3, 2.0.0 and 2.1.0.

 * Debian 7
 * EL 5
 * EL 6
 * EL 7
 * SLES 10
 * SLES 11
 * SLES 12
 * Ubuntu 12.04 LTS
 * Ubuntu 14.04 LTS
 * Solaris 9
 * Solaris 10
 * Solaris 11
